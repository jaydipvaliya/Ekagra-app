# Ekagra - AI-Powered Productivity App

## 🌟 Overview

Ekagra is a comprehensive productivity application designed to help you achieve deep focus, build lasting habits, and accomplish your goals. Built with React and TypeScript, it combines the Pomodoro Technique with habit tracking, goal setting, and motivational tools to create an optimal environment for productivity.

The app features an integrated AI assistant powered by Google's Gemini API to provide personalized motivation, task suggestions, and productivity insights.

## ✨ Features

### 🕒 Pomodoro Timer
- Customizable focus sessions (default 25 minutes)
- Short breaks (5 minutes) and long breaks (15 minutes)
- Deep work mode for extended focus periods
- Auto-start next session option
- Sound notifications

### 📝 Task Management
- Eisenhower Matrix for task prioritization
- Daily task planning with calendar integration
- Pomodoro estimation and tracking
- Task status management (Active, Done, Cancelled)

### 🎯 Habit Tracking
- Daily habit completion tracking
- Visual progress indicators
- Streak maintenance
- Habit creation and management

### 📅 Calendar View
- Daily task overview
- Date-based task filtering
- Goal countdown visualization

### 🎵 Ambience & Motivation
- Ambient soundscapes for focus
- Curated motivational quotes
- Embedded productivity videos
- Daily intention setting
- Personalized encouragement messages

### ⚙️ Customization
- Dark/Light theme support
- Customizable timer durations
- Sound preferences
- Goal setting with countdown timers

### 🤖 AI Integration (Coming Soon)
- Personalized productivity recommendations
- AI-generated motivational content
- Smart task prioritization suggestions
- Progress analysis and insights

## 🚀 Installation

### Prerequisites
- Node.js (version 16 or higher)
- npm or yarn package manager

### Setup
1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd ekagra-app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Environment Configuration:**
   Create a `.env.local` file in the root directory and add your Gemini API key:
   ```
   GEMINI_API_KEY=your_gemini_api_key_here
   ```
   > **Note:** Get your API key from [Google AI Studio](https://ai.google.dev/)

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to `http://localhost:5173` (or the port shown in your terminal)

## 📖 Usage

### Getting Started
1. **Set Your Daily Intention:** In the Motivation tab, write down what you want to accomplish today.
2. **Plan Your Tasks:** Use the Day Planner to add tasks with estimated Pomodoro sessions.
3. **Prioritize:** Move tasks to the Eisenhower Matrix to categorize by importance and urgency.
4. **Start Working:** Begin your first Pomodoro session and maintain focus.
5. **Track Habits:** Mark off daily habits in the Habit Tracker.
6. **Set Goals:** Create long-term goals with countdown timers.

### Key Workflows
- **Pomodoro Flow:** Add tasks → Start timer → Work → Break → Repeat
- **Habit Building:** Create habits → Daily check-ins → Track streaks
- **Goal Achievement:** Set goals → Daily progress → Celebrate milestones

## 🔧 Configuration

### Timer Settings
Customize your Pomodoro sessions in the Settings modal:
- Focus duration (default: 25 minutes)
- Short break (default: 5 minutes)
- Long break (default: 15 minutes)
- Deep work session (default: 90 minutes)
- Rounds before long break (default: 4)

### Audio Preferences
- Enable/disable sound notifications
- Adjust ambience sound volumes
- Choose from various ambient soundscapes

### Theme
Toggle between light and dark themes for comfortable extended use.
